# E-Commerce Analytics Project

## Overview
End-to-end data analytics project analyzing 5,000 e-commerce orders using Python.

## Key Findings
- 85% delivery success rate
- Electronics is the top revenue category
- 50% of customers pay by Credit Card
- Average customer rating: 4.2/5.0
- ML model predicts customer satisfaction successfully

## Technologies Used
- Python, Pandas, Matplotlib, Seaborn
- Scikit-learn (Random Forest ML Model)
- Jupyter Notebook

## Project Structure
- data/ — Dataset and chart images
- notebook/ — Jupyter notebooks
